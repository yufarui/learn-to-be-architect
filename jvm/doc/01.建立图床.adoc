== 画图工具的选择-draw.io

- 使用draw.io;将 https://www.draw.io 绑定至github
- 使用 https://picx.xpoet.cn/[picx插件网址]
- 在github中设定自己的token,在picx官网中输入此token
- picx会帮我们快速创建cdn缓存图片
